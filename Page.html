Learning Analytics Package Generator

This small web app creates for you a package with all that you need to run a learning-analytics-enhanced session to run with your students

View the Project on GitHub

Create Your Session Package

1. Your email

So that we can share with you the Google Docs, Forms and Spreadsheets used to manage and gather data during the session. Please use your @gmail.com or other Google domain address (e.g., @tlu.ee).

2. Give the session a distinctive name

Try to make it unique (e.g., include the course/subject, today's date, where it will happen...).

3. Choose what kind of session it will be

Interactive Feedback Session

Students present their work in short presentations, gather feedback about them in short forms, and finally breakout groups are formed for more detailed feedback and discussions about each of the presentations. Also very useful for scientific conferences!

Living Labs Example

A simple session to illustrate how learning process and other psychological construct questions can be embedded into an existing package developed by teachers and researchers.
